## 1. 戦略コンテキスト（必読）

### 1-1. ケンミツの事業概要

「ケンミツ」は **建設業（中小・一人親方）向けの見積書クラウドサービス**。
本サービスの差別化価値は **改正建設業法 2025（2025/12/1 全面施行）に対応した
見積書を低価格で出せる**こと。同様の機能を持つ大手 SaaS（ANDPAD・現場ポケット 等）
は月¥9,800〜数万円の価格帯。ケンミツは月 ¥1,980 で同等の改正法対応をカバーする。

### 1-2. Phase 1 ピボット（2026-04-25・commit `40000fe`）で何が変わったか

旧モデル（〜2026-04-24）:
- Free: 透かし常時 ON + 月 3 通保存制限
- Solo: ¥980/月、透かしなし・無制限
- 課金主ドライバー: **透かし**（取引先に出す時に必要になる）
- ファネル: 広告 → LP → 直接課金（短期）

新モデル（2026-04-25〜）:
- Free: **完全無料・無制限**（透かし廃止・保存制限廃止）
- 有料: **¥1,980/月**（年 ¥19,800）、改正建設業法対応版 PDF + チェッカー + マスタ + 原価分析 + 工事写真
- 課金主ドライバー: **改正建設業法対応版 PDF**（労務費・法定福利費の内訳明示）
- ファネル: 広告 → /start で登録 → ツール使用 → ドリップメール教育 → 有料転換（時間差）
- メアド登録必須化（旧「登録不要で試せる」訴求は撤回）

### 1-3. ピボットの背景にある事業判断

**(a) 広告キーワード戦略の発見（2026-04-22〜25 の初回出稿で判明）**

- 「見積書」系キーワード（「見積書 無料」「建設業 見積書」等）は **検索ボリュームが
  あり、CPC が安く取れる**
- 「改正建設業法」系キーワードは **検索ヒットが薄い**（業界従事者でも能動的に
  検索しないテーマ）
- 結果: 改正法系で広告を出しても集客できないが、見積書系で広告を出すと CPA が
  健全な水準に収まる可能性が高い

**(b) 競合分析**

- 「無料の見積書ツール」は他社多数存在（Misoca / 見積書テンプレ Word/Excel /
  汎用 SaaS の無料プラン）
- 単に「無料で見積書」という軸では差別化できない
- ケンミツの本当の差別化価値は「**改正建設業法対応**」。だがこれは知識前提で、
  能動的にキーワード検索しないユーザーが多い

**(c) 結論として導いた戦略**

> 「見積書 無料」で**集客**して、ツール内の体験で**改正法対応の必要性に気づかせる**。
> 教育には時間がかかるので**ドリップメールでアナウンス**。だからメアド登録が必須。
> 登録時点では課金訴求しない。Free を完全無料化することで登録率を最大化する。

つまり「Free を完全無料化」は損ではなく、**メールリスト構築のための投資**。
LTV は時間差ファネルで回収する設計。

### 1-5. なぜ単発 CV ではなく LTV で見るべきか

ピボット後のファネルでは、登録から課金まで**最短 5 日・通常 1〜数週間**かかる。
広告出稿初日に課金 CV ゼロでも諦めない。月単位で:
- 登録数（メアド取得数）
- 登録 → 課金転換率（業界水準 1〜5%）
- 月額 ¥1,980 × 平均継続月数（=LTV）
- LTV ÷ 登録 1 件あたり広告費 = 投資効率

短期 CPA 重視の判定は誤判断を生む。

### 1-6. 価格ポジショニング（¥1,980 の合理性）

| サービス | 月額 | カバー範囲 |
|---|---|---|
| 大手建設業 SaaS（ANDPAD / 現場ポケット 他） | ¥9,800〜数万円 | 業務管理フル装備 |
| 会計ソフト（freee / マネーフォワード / やよい） | ¥1,180〜2,380 | 個人事業主向け基本機能 |
| **ケンミツ** | **¥1,980** | 改正建設業法対応の見積書特化 |

会計ソフト個人事業主プランと同価格帯で、建設業特化の専門性を提供。即決圏内
（¥2,000 までは「考えずに払える」心理ライン）。

### 1-7. 前回広告（4/22-25）の失敗から学んだこと

4 日間（¥4,739 消化・CTR 6.91% に見えた）の Search Terms レポート分析で判明:

- **「インテントマッチ」でターゲット外配信**: 完全一致 KW でも Google AI が
  意味拡張し、「リフォーム見積書」「グリーンサイト」「土木 積算ソフト」「家
  建てる」等に配信されてた
- 改正建設業法系 KW では検索ヒットゼロ（需要が薄い）
- CTR 6.91% は粉飾、実 CV はゼロ

→ 新戦略への反映:
- **CPA 安く取れる「見積書 無料」系を主軸**にして、改正建設業法系は補助
- 完全一致 + フレーズ一致のみ（部分一致禁止）
- 強力な除外 KW で意図しない拡散を防ぐ
- 単発 CV ゼロは判定材料にせず、メール経由の時間差転換を待つ

## 2. キャンペーン構成

### キャンペーン A: 「無料の見積書作成ツール」（メイン・予算 80%）

| 項目 | 設定 |
|---|---|
| 目的 | **/construction/start にクリックを最大化 → Free 登録（メアド取得）** |
| 月予算 | ¥8,000（日 ¥267） |
| 入札戦略 | **クリック数最大化**（CPC 上限 ¥200） |
| 着地ページ | `https://mitsumori-maker.com/construction/start` |
| マッチタイプ | **完全一致 + フレーズ一致**（部分一致は禁止・前回失敗の主因） |
| 主要 KW | 過去（4/22-25）の Search Terms で実際にクリックが発生した KW を中核に置く |
| デバイス | PC + モバイル両対応（建設業は両方使う） |
| 配信地域 | 日本全国 |

**入札戦略の判断（重要）**:

「コンバージョン数の最大化」ではなく **「クリック数の最大化」**を採用する理由:
- ゴールは「無料会員数を増やす」 = 着地ページに大量に流す方が直接的
- コンバージョン最適化は CV データが蓄積するまで（数十件〜）配信が安定しない。
  予算 ¥8,000/月で CV 月数件規模だと最適化が収束せず、配信が絞られるリスク
- クリック数最大化なら CPC ¥100〜200 で月 40〜80 クリックを狙える
- CPC ¥150 平均 × 60 クリック × 登録率 40%（/start は登録特化なので高い見込み）
  = 月 24 件の登録 → CAC 約 ¥333

CV 計測は引き続き行うが、**最適化指標としては使わない**。十分な CV データが
蓄積した段階（3 ヶ月〜半年後）で「コンバージョン数最大化」への切替を再評価。

### キャンペーン B: 「改正建設業法 教育」（補助・予算 20%）

| 項目 | 設定 |
|---|---|
| 目的 | 改正法を学びたい層の獲得（一応押さえておく程度） |
| 月予算 | ¥2,000（日 ¥67） |
| 入札戦略 | **クリック数最大化**（CPC 上限 ¥200・A と統一） |
| 着地ページ | `https://mitsumori-maker.com/construction/guide/kaisei-kensetsu-2025` |
| マッチタイプ | **完全一致のみ**（フレーズ一致も拡散リスクあり） |
| 主要 KW | 「改正建設業法 2025」「改正建設業法 見積書」「労務費 内訳明示」「建設業法 法定福利費」「建設業法 改正 2025」「建設業 一式記載」 |

**B の位置づけ**:
- 予算は最小限（A で月 ¥10,000 のうち 20%）
- 改正建設業法系は前回 Search Terms でクリックゼロ確認済み（需要薄）
- 「一応押さえておく」レベルで配信、表示が出ない場合は予算を A に寄せる
- 入札戦略は A と同じくクリック数最大化（CV 最適化は予算規模的に収束しない）

### 予算合計

- 月 **¥10,000**（C1: ¥7,000 + C2: ¥3,000）
- 5 日間で約 ¥1,667 消化想定
- 月末に ¥10,000 着地、その時点で判定

## ユニットエコノミクスと撤退基準

### 粗利の前提
- 実質粗利 ¥1,800〜¥1,900/月（Stripe 手数料 + 固定費控除後）
- 健全 CAC 上限: **¥10,000 以下**を目標、¥8,000 以下が理想

### 撤退ライン（必読）

**月¥10,000 消化でサブスク 0 件 → 撤退検討**

ただし以下のサブ基準で判断を分ける:

| 状況 | 判定 |
|---|---|
| 登録 CV も 0〜数件 | **即撤退**（着地・KW・広告コピーが機能していない） |
| 登録 CV 10 件以上・課金 0 件 | **延長 +1 ヶ月**（前段機能・課金転換のみの問題） |
| 登録 CV 5 件前後・課金 0 件 | **半月延長 +¥5,000**（微妙ライン） |

「サブスク 0 件」単独で判断しないこと。**時間差ファネル**（広告 → 登録 → Day 5
メール → 検討 → 課金）の特性上、月末登録分の課金は翌月になる。登録 CV が出ていれば
ファネル前段は機能している証拠で、課金転換率改善の余地がある。

### 改善試行の選択肢（撤退前に試せる）

登録は出るが課金が出ない場合:
1. ドリップメール Day 5 の文言 A/B
2. Day 10 / Day 18 / Day 28 メールを追加（現状 Day 0/2/5 のみ）
3. エディタ内の課金 CTA 強化（PDF DL 完了モーダルの初回大 CTA は実装済）
4. /construction/start に小さく改正法訴求を入れる（教育の前倒し）

2 ヶ月目もダメなら戦略変更 or 撤退。

## 公開後の運用ルール（私の作業）

- **5 日間は触らない**（変数固定で判定精度維持）
- 5 日経過時点で:
  - Search Terms レポート確認 → 想定外 KW があれば除外追加

LP は「**既登録ユーザー向けの有料プラン課金訴求ページ**」（commit `a429981` で再定義済み）。
